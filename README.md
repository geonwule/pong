Linux & MacOS:

```sh
make
```

## 실행

Server:

```sh
./pong_server [ip_address(생략시 기본값 = localhost)] <port_number> 
```


Client:

- run
   ```sh
   ./pong_client [ip_address(생략시 기본값 = localhost)] <port_number>
    ```
- game start
   ```
   Game Start
   ```
